Connected Accounts

Multiply your quota by rotating across accounts.

Rotation pool

Two roles, like the pros. Uploaders (Open Cloud key) take turns uploading - Roblox's audio quota is per account (100/month), so when one runs out AssetFlow moves to the next and every asset still lands in your game. Downloaders (cookie) take turns fetching - different accounts can access different assets, so adding alts reaches more.

Add an account

Uploader: fill creator id + Open Cloud key (asset:write). Downloader: paste just a cookie (leave key/creator blank). An account with both does both.

Setup & FAQ

Get running in a few steps, with real screenshots.

Step-by-step

From zero to a working reupload in about two minutes.

1

Create an Open Cloud API key

Sign in at create.roblox.com, open Open Cloud → API Keys, and click Create API Key.

2

Name it and add permissions

Give it a name. Under Access Permissions add the Assets API with asset:read and asset:write, then add Asset Permissions with asset-permissions:write. Leave Restrict by Creator OFF.

3

Set security and generate

Under Security, set Accepted IP Addresses to 0.0.0.0/0, then click Save & Generate Key.

4

Copy your key

Copy the key now. Roblox shows it only once. Keep it private and never share it with anyone.

5

Add it to AssetFlow and set your target

Paste the key into Credentials on the Dashboard and Save. Choose My profile or A group (enter the group id), then Save. Pool several keys under Accounts to beat the monthly quota.

6

Install the plugin and reupload

Click Install plugin to drop it into your Studio plugins folder automatically, then restart Studio. (Manual way: copy AssetFlowReuploader.rbxmx into %LOCALAPPDATA%\Roblox\Plugins.) Then open the AssetFlow Reuploader plugin, pick Animation / Audio / Image, and hit Reupload. New ids swap into your place automatically.

Your API key gives access only to the permissions you pick. Never share it. AssetFlow stores it only on this machine and sends it only to Roblox, it is never shared with anyone.

FAQ

Why does audio sometimes stop partway?
Roblox caps audio uploads at 100/month per account. AssetFlow detects the cap and rotates to your next account automatically. Add more under Accounts to keep going.
Do I need the cookie?
Yes - every type (animations, sounds, images, meshes) is pulled from the asset CDN using your cookie, so a valid cookie is required. Note: logging into Roblox in your browser expires the saved cookie, so re-paste it if reuploads start failing.
Is my account safe?
Your API key and cookie are stored only on this machine and are never sent anywhere except Roblox itself. Uploads use Roblox's official Open Cloud API.
Why did an asset get skipped?
With "skip owned" on, anything already owned by your target account or group is left alone so you don't waste quota. Turn it off to force a fresh copy.
Where do the new ids go?
They're swapped in automatically across instances, image properties, value objects, attributes, and script source, all under one Undo in Studio.
Can I add more than one account?
Yes. Add several under Accounts and AssetFlow rotates to the next when one hits its monthly quota, so you can keep going.
